## git init

**git init** это команда создаёт в текущей директории новую поддиректорию с именем .git, содержащую все необходимые файлы репозитория — структуру Git-репозитория.

По сути, эта команда создает пустой репозиторий на вашем компьютере.


```-bash
git init
```
